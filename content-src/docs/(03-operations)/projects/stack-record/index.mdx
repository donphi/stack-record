## What this project is

Stack Record is the knowledge management system you are reading right now. Built on Fumadocs, it captures concepts, methods, systems, decisions, experiments, and projects as a structured, navigable knowledge base.

## Why it exists

Research and engineering knowledge decays when it lives in scattered notes and tribal memory. Stack Record applies the [[closure_rule]] — every referenced idea gets its own note — to build a self-contained, auditable knowledge graph.

## Scope

- Structured content authoring in MDX with [[metadata_schema]] validation
- Automated knowledge graph generation from wiki-link cross-references
- Pipeline integration via [[rabbitmq_central_broker]] for content events
- Navigation maps, domain indexes, and search across all content layers

## Status

Active. The documentation framework is live; content is being populated and the KG builder pipeline is under development.
