## What this project is

Gatehouse is a data governance and access control system for the knowledge pipeline. It manages who can read, write, and transform data as it flows through extraction, embedding, and retrieval stages.

## Why it exists

As the pipeline handles sensitive biomedical data and third-party API credentials, a dedicated governance layer is needed to enforce access policies, audit data lineage, and satisfy compliance requirements.

## Scope

- Credential vaulting and rotation for pipeline services
- Role-based access control for data stores and API endpoints
- Audit trail generation for data transformations
- Policy-as-code enforcement at pipeline boundaries

## Status

Draft. Architecture is under design; no production deployment yet.
