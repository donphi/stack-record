## What this map is

System notes on security controls: credential management, access control, and supply-chain integrity for the knowledge pipeline.

## Why this matters

The pipeline handles API keys, model weights, and potentially sensitive biomedical data. A single leaked credential or unpatched dependency can compromise the entire system.

## Where this fits

Security systems constrain every other layer: [[runtime]] for secret injection, [[build]] for supply-chain verification, [[data]] for access control, and [[pipeline]] for audit trails.
