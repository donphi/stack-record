## Getting Started

This section covers the basics of setting up your environment.

### Prerequisites

You will need Node.js 18+ and a package manager like pnpm.

### Installation

Run the following command to install dependencies:

```bash
pnpm install
```

### Configuration

Create a config file in the root of your project.

## Architecture

An overview of how the system is structured.

### Core Modules

The core is split into three layers: data, logic, and presentation.

### Data Flow

Data flows unidirectionally from sources through transformers to views.

## Deployment

How to ship your project to production.

### Staging

Push to the staging branch to trigger a preview deployment.

### Production

Merge to main to deploy to production automatically.

### Rollback

If something goes wrong, revert the last merge commit.
