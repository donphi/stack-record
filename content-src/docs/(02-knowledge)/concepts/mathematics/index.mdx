## What this map is

This map is the entry point for mathematical concepts used across the knowledge base: vectors, linear algebra, set theory, and related foundations. It groups definitions and worked intuitions so you can jump from a symbol or structure to the right note without hunting the whole tree. Treat it as the conceptual spine for anything that looks like geometry, algebra, or discrete structure.

## Why this matters

Many downstream ideas—similarity, optimisation, probability on spaces—lean on the same few mathematical objects. When those objects are named and linked consistently, arguments stay checkable and reuse stays cheap. A clear mathematics map also makes gaps obvious before they block a derivation or implementation.

## Where this fits

This domain sits under broader knowledge maps and feeds methods (e.g. retrieval, ML), systems (e.g. numerical pipelines), and governance notes that reference formal claims. Links here should point to small, single-purpose notes rather than long tutorials.
